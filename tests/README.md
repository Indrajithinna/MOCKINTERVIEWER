# Tests

This directory is reserved for unit and integration tests.

## Backend Tests
To create backend tests:
1. Create a `MockInterview.Tests` project.
2. Add reference to `MockInterview.API`.
3. Use xUnit or NUnit.

## Frontend Tests
To create frontend tests:
1. Use Vitest or Jest.
2. Add `*.test.tsx` files in `frontend/src/tests`.
